This rule checks that logic layer does not depend on service layer. Allowed dependencies from the logic layer:

  • Logic layer of a different component
  • Dataaccess layer of the same component
  • Common layer

Noncompliant Code Example

    package com.devonfw.ide.sonarqube.logic.api;

    import com.devonfw.ide.sonarqube.service.api.Bar;
    
    public class MyClass {}
